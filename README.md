# Creative Portfolio

Experimental avant-garde portfolio inspired by Sagmeister's bold, anti-intuitive design philosophy.

## Features

- Bold color collisions & dynamic typography
- Anti-conventional layout that breaks the grid
- Strong visual impact with intentional chaos
- 5 content categories: Graphic Design, Video, Copywriting, Planning, Bio

## Tech Stack

- Single HTML file (vanilla CSS + JS)
- No build tools required
- Responsive design

## Quick Start

Open `index.html` in any modern browser:

```bash
# or simply
start index.html
```

## Content Structure

```
assets/
├── graphic design/    # 平面设计作品
├── photography works/  # 摄影作品
├── video/            # 视频作品
└── xhs/              # 小红书文案案例
```
